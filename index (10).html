Firebase Kurulumu

Google ile giriş için Firebase Authentication kurmanız gerekiyor:

  1. console.firebase.google.com adresine gidin
  2. Yeni proje oluşturun
  3. Authentication → Sign-in method → Google'ı etkinleştirin
  4. Project Settings → Firebase SDK snippet → Config'i kopyalayın
  5. Bu HTML dosyasının sonundaki firebaseConfig nesnesini doldurun
  6. E-posta/şifre girişi için Authentication'dan Email/Password'ü etkinleştirin

Firebase kurulmadan demo mod çalışmaya devam eder — API key girerseniz gerçek yanıtlar alırsınız.
